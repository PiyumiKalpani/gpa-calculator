# Advanced GPA Calculator

## 📌 Project Overview
The Advanced GPA Calculator is a web-based application developed using HTML, CSS, and JavaScript. It helps students calculate their Grade Point Average (GPA) accurately based on course grades and credit values. The system is designed to be user-friendly and efficient for academic performance tracking.

## 🚀 Features
- Add multiple subjects with grades and credits
- Automatic GPA calculation
- Support for different grading systems
- Dynamic input fields (add/remove subjects)
- Instant result display
- Clean and responsive user interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- `index.html` – Main interface of the GPA calculator  
- `style.css` – Styling and layout design  
- `script.js` – GPA calculation logic  

## 🎯 Purpose of the Project
This project was developed to demonstrate front-end development skills and provide a practical tool for students to easily calculate and monitor their GPA.

## 📊 How It Works
- Users enter subject names, grades, and credit values  
- The system calculates total grade points  
- GPA is calculated automatically using the formula:  

GPA = Total Grade Points / Total Credits  

## ⚙️ How to Run the Project
1. Download or clone the repository  
2. Open the project folder  
3. Open `index.html` in any web browser
